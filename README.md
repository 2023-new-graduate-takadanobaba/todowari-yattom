# 001 数字にする
ユーザーが入力した文字列を数字に変換する。ただし、ユーザーが入力した文字列が数字でない場合は、"数字ではありません"と表示する。

## 具体例

- case 1
  - input
    - 3
  - output
    - 3 
- case 2
  - input
    - 909
  - output
    - 909
- case 3
  - input
    - abc
  - output
    - 数字ではありません
